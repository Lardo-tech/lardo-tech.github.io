Mijn programmeer ervaring

op mijn middelbare school het Ostrea Lyceum heb ik een klein beetje programmeer ervaring op gedaan. Zo heb ik al een beetje gewerkt met HTML, CSS, Python en javascript. Elke periode/kwartaal deden we een andere opdracht met weer een andere programmeer taal.

De cursussen die we hadden gedaan op het Ostrea waren allemaal via Code Avengers. Tijdens de informatica lessen op het Ostrea kwam ik er achter dat ik het erg leuk vond om programmaatjes te maken. Verder vond ik het ook interessant om te zien hoe je een computer op een bepaalde manier moet aan sturen om je doel te kunnen bereiken. De cursussen op Code Avengers waren erg saai en kinderlijk in gericht, waardoor het soms moeilijk was om je aandacht er bij te houden. De opdrachten werkten echter wel en als Code Avengers af was kregen we opdrachten van de docent om dan met die programmeer taal uit te gaan voeren.
